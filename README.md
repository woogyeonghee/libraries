# the way to build opencv project


goto ur project dir
~~~
cd {OpenCV_INCLUDE_DIRS}
~~~


make dir named build dir
~~~
mkdir build
~~~


goto build dir
~~~
cd build
~~~


cmake
~~~
cmake ..
~~~


make
~~~
make
~~~


execute ur program
~~~
./a.out
~~~
